#### management commands

##### createinitialrevisions

```sh
manage.py createinitialrevisions

manage.py createinitialrevisions \
    your_app.YourModel \
    --comment="Initial revision."

manage.py createinitialrevisions --help  # for more info
```

<small>

For large databases, this command can take a long time to run.

</small>
