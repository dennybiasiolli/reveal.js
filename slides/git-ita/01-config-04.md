### Chiedere aiuto

Se dovessi avere bisogno di aiuto durante lâ€™uso di Git, ci sono due modi per visualizzare la pagina di aiuto di ciascun comando di Git, direttamente dal manuale.

```sh
git help <comando>
git <comando> --help
```

Puoi, per esempio, leggere la pagina di manuale relativa al comando `init`

```sh
git help init
```
