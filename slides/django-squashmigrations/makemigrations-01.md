#### 🧑‍🔧 `makemigrations`

Creates new migration(s) for apps.

```sh
manage.py makemigrations [--empty] [-n NAME] [app_label]
```

- `--empty`<br>
  Create an empty migration.

- `-n NAME, --name NAME`<br>
  Use this name for migration file(s)

- `app_label`
