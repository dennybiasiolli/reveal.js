#### Slot Content and Outlet

1. allow a component to accept dynamic content

    ```html
    <button class="fancy-btn">
        <slot></slot> <!-- slot outlet -->
    </button>
    ```

2. pass a template fragment to a child component

    ```html
    <FancyButton>
        Click me! <!-- slot content -->
    </FancyButton>
    ```
