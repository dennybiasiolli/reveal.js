#### Input types

- &lt;input&gt; with text types and &lt;textarea&gt;

  :value property and @input event

- &lt;input type="checkbox"&gt; and &lt;input type="radio"&gt;

  :checked property and @change event

- &lt;select&gt;

  :value as a prop and @change as an event
